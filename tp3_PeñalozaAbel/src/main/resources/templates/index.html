Objetivo de la Materia

Materia de segundo año de la carrera Analista Programador Universitario. Donde el alumno aprendera a desarrollar aplicaciones web de gestión de tareas (como un gestor de tareas personales o de proyectos) utilizando Java con Spring Boot, que incluya diferentes funcionalidades.
